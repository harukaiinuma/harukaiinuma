# GitHubの練習

このリポジトリはMs.Engineerのプレゼンテーションチェックインの中にある課題です。


## <span style="color: purple; ">これからの目標</span>
* **子供の成長をサポートできる環境づくり**
* **自分に自信をつける**

##　<span style="color: purple; ">その目標のために行うこと</span>
**[自己成長する]**  
エンジニアになるために学習をし、　***自分が実現したい働き方***を手にいれる。  
お金について学習をし、***家族を守るスキル***を手にいれる。


**[自分について知る]**  
***自己主張***できるようになる。  

今までは周りを気にしすぎで自分がどうしたいかということを無視して生活をしていた。学生時代は人間関係良好、部活ではキャプテン、学校では学級員長を任されたり、先生にも気に入られていた。社会に出てからは個性がなく、人の顔色ばかりうかがい、否定できない、お願いできない、Helpを出せない状況で苦しんだ。現在も自己主張やHelpを出すことは苦手だが、ここを打破したい。そのために日記を書き、自己会話をしている。

**[子供を観察すること]**  
子供たちが ***自分の力で幸せになる*** 方法を見つけ出す。

子供が今”何をしたいのか”、”どう思っているのか”できるだけ観察しながら子育てをしたい。助けすぎず、先回りしすぎず、会話と観察でサポートし、子供たちが自分の力で幸せになる方法を見つけ出してほしい。
